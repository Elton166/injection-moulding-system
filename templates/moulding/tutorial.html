1️⃣ Getting Started (5 minutes)

📝 Step-by-Step Guide: Company Registration & First Login
Step 1: Register Your Company
  1. Open the app in your browser
  2. Click the "Register Company" button
  3. Fill in:
    • Company Name (e.g., "ABC Manufacturing")
    • Email (e.g., "info@abcmfg.com")
    • Phone Number
    • Address
    • Username (for login)
    • Password (secure password)
  4. Click "Register"
  5. ✅ You're now logged in as the company!
Step 2: Create Employee Accounts
  1. From your company dashboard, click "Register Employee"
  2. Select role: Manager, Supervisor, or Operator
  3. Fill in employee details (name, email, username, password)
  4. Click "Register"
  5. ✅ Employee can now login with their credentials!
Step 3: Navigate the Dashboard
  • Active Moulds: Shows how many moulds are running
  • Pending Orders: Orders waiting to be produced
  • Open Issues: Problems that need attention
  • Housekeeping: Cleaning tasks status
  • Quick Actions: Buttons for common tasks

2️⃣ Production Management (8 minutes)

📝 Complete Production Workflow
Part A: Create a Production Order
  1. Click "Production Orders" in menu
  2. Click "Create New Order"
  3. Fill in order details:
    • Product Name (e.g., "Plastic Container")
    • Customer Name (e.g., "XYZ Corp")
    • Quantity (e.g., 10000)
    • Unit (e.g., "pieces")
    • Priority: Urgent/High/Normal/Low
    • Due Date
    • Select Mould
    • Select Material Recipe (optional - for auto-calculation)
  4. Click "Create Order"
  5. ✅ Order appears in dashboard with priority badge!
Part B: Start a Mould Run
  1. From dashboard, click "Start New Run"
  2. Select the mould you're using
  3. Enter machine number
  4. Enter setter name
  5. Set start time and setter completion time
  6. Add notes (optional)
  7. Click "Start Run"
  8. ✅ Run appears on dashboard with live duration timer!
Part C: Record Shift Production
  1. Click "Shift Production" in menu
  2. Click "Record Production"
  3. Select the production order
  4. Select shift (Morning/Afternoon/Night)
  5. Enter shift date and times
  6. Enter quantities:
    • Quantity Produced (good parts)
    • Quantity Rejected (scrap)
  7. App automatically calculates:
    • Scrap Rate %
    • Production Rate (parts/hour)
    • Efficiency %
  8. Click "Record"
  9. ✅ Supervisor can now approve the production!
Part D: Stop Mould Run
  1. Go to dashboard
  2. Find the active run
  3. Click "Stop Run" button
  4. ✅ Run is marked complete with total duration!

3️⃣ Material Management (6 minutes)

📝 Complete Material Setup Guide
Part A: Add Material Types
  1. Click "Materials" in menu
  2. Click "Material Types"
  3. Click "Add Material Type"
  4. Fill in:
    • Material Code (e.g., "PP-V001")
    • Name (e.g., "Polypropylene Virgin")
    • Category: Virgin/Regrind/Mixed
    • Supplier
    • Current Stock (kg)
    • Minimum Stock (for alerts)
    • Cost per kg
  5. Click "Save"
  6. ✅ Material added! Low stock alerts will show if stock is low
Part B: Add Masterbatch Colors
  1. From Materials menu, click "Masterbatch"
  2. Click "Add Masterbatch"
  3. Fill in:
    • Code (e.g., "MB-RED-001")
    • Name (e.g., "Red Masterbatch")
    • Type: Color or Additive
    • Color Hex Code (e.g., #FF0000 for red)
    • Supplier
    • Current Stock
    • Minimum Stock
  4. Click "Save"
  5. ✅ Color appears with visual color swatch!
Part C: Create Material Recipe
  1. Click "Material Recipes"
  2. Click "Create Recipe"
  3. Fill in:
    • Recipe Code (e.g., "RCP-001")
    • Recipe Name (e.g., "Red Container Mix")
    • Select Base Material
    • Base Material % (e.g., 85%)
    • Select Masterbatch
    • Masterbatch % (e.g., 5%)
    • Regrind % (e.g., 10%)
  4. Click "Save"
  5. ✅ Recipe ready! Use it in production orders for auto-calculation
💡 Pro Tip: When you create a production order and select a material recipe, the app automatically calculates how much of each material you need (including 5% waste factor)!

4️⃣ Quality Control (7 minutes)

📝 Quality Management Workflow
Part A: Submit Hourly Checklist
  1. Click "Checklists" in menu
  2. Click "Submit Checklist"
  3. Fill in:
    • Machine Number
    • Mould being used
    • Temperature readings
    • Pressure readings
    • Cycle time
    • Check "Issues Found" if problems detected
    • Describe any issues
  4. Click "Submit"
  5. ✅ Checklist saved! Appears on dashboard
Part B: Report an Issue
  1. Click "Issues" in menu
  2. Click "Report Issue"
  3. Select category:
    • Customer Complaint
    • Mould Issue
    • Product Defect
    • Machine Issue
    • Maintenance
  4. Fill in:
    • Title (brief description)
    • Detailed description
    • Priority: Critical/High/Medium/Low
    • Related mould/machine/customer
  5. Click "Report"
  6. ✅ Issue created with tracking number!
Part C: Use Troubleshooting Database
  1. Click "Troubleshooting" in menu
  2. Browse 17 common issues:
    • Short Shots
    • Flash
    • Sink Marks
    • Warpage
    • And more...
  3. Click on an issue to see:
    • Symptoms
    • 10-step solution
    • Machine parameter adjustments
    • Prevention tips
  4. ✅ Follow the steps to fix the problem!
Try It Now: Report Issue

5️⃣ Housekeeping (4 minutes)

📝 Housekeeping Task Workflow
Complete Housekeeping Process:
  1. Click "Housekeeping" in menu
  2. Click "Start Task"
  3. Fill in:
    • Task description
    • Area type: Machine/Mould/Factory Floor/Storage/Workstation
    • Specific area (e.g., "Machine 5")
  4. Take "Before" Picture: Upload photo of area before cleaning
  5. Click "Start Task"
  6. ✅ Task created and marked "In Progress"
  7. Perform the cleaning
  8. Click on the task to complete it
  9. Click "Complete Task"
  10. Fill in:
    • Take "After" Picture: Upload photo after cleaning
    • Cleaning products used
    • Any issues found during cleaning
    • Completion notes
  11. Click "Complete"
  12. ✅ Task marked complete with before/after photos!
💡 Pro Tip: Before/after pictures provide visual proof of cleaning and help maintain standards!

❓ Frequently Asked Questions

Q: How do I reset my password?

A: Contact your company administrator to reset your password.

Q: Can I edit a production order after creating it?

A: Yes, go to Production Orders, click on the order, and use the edit option.

Q: What happens if I forget to stop a mould run?

A: You can stop it anytime from the dashboard. The duration will be calculated from start to stop time.

Q: How do I know if materials are running low?

A: The Material Dashboard shows low stock alerts when stock falls below minimum levels.

Q: Can I access the app on my phone?

A: Yes! The app is fully responsive and works on phones, tablets, and computers.
